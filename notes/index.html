Constrained Hamiltonian Dynamics

This concerns classical mechanics systems which are dictated to obey constraints. In some such cases, some problems occur when promoting the Poisson brackets of classical mechanics to commutators of quantum mechanics. These are some old notes regarding that.

Monte Carlo simulations of Ising model

This reports some basic Monte carlo simulations of the 2D Ising model. A brief description of Monte Carlo methods, Metropolis alogrithm, finite size scaling and an implementation of an algorithm for the data-collapse to find scaling exponents can be found here. Chapter one concerns a biological motivation for this project and can be skipped safely.

Site percolation on the Bethe lattice

One page note on finding the critical \(p\) for percolation on a Bethe Lattice with \(z=3\) where \(z\) is the number of neighbors. Using a self-consistency condition, we can also find the probability of a site belonging to the percolating cluster and consequently the critical exponent for the continuous transition of percolation. Bethe lattices make calculations simpler due to the feature that the lattice does not have loops.
